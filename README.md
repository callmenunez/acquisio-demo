Sales Demo Pages
==========

Installation
------------

1. `git clone TBD`
2. `cd` into the directory
3. Run `npm i && bower i`
4. Add

`127.0.0.1 acquisiodemo.fed`

to /etc/hosts

5. Once the packages are installed, run `gulp` to start the server
6. Go to [http://acquisiodemo.fed:3008](http://acquisiodemo.fed:3008)
